# HR Analytics: Job Change Prediction

## 📊 Project Overview
This project analyzes HR data to understand factors that influence whether a data scientist changes their job or stays in the same company.

The dataset is from Kaggle HR Analytics competition.

---

## 🎯 Objective
- Analyze employee characteristics
- Identify factors affecting job change
- Build insights using Python and Pandas

---

## 🧰 Tools Used
- Python
- Pandas
- Matplotlib
- GitHub

---

## 📁 Dataset
The dataset contains information about:
- Education level
- Experience
- City development index
- Training hours
- Target (Job change: 0 = stay, 1 = change)

---

## 📊 Key Insights
- Distribution of employees who changed job vs stayed
- Relationship between experience and job change
- Impact of training hours on job change

---

## 📌 Files in this repository
- `analysis.py` → main analysis code
- `data/` → dataset files
- `images/` → generated charts
